## 📊 コンテンツ戦略

### ブログ記事の作り方

**トピックの選び方**:
1. 検索需要のあるトピックを探す
2. 競合が少ないニッチトピックを選ぶ
3. 自分の経験を活かせるトピックを優先

**記事の構成**:
1. キャッチーなタイトル（検索意図を含む）
2. 簡潔でわかりやすいリード文（150〜200字）
3. 見出しやすい見出し（箇条書き、太字、番号）
4. 関連性のあるH2見出し（3〜5つ）
5. 各H2見出しの下に400〜800字の本文

**E-E-A-Tの原則**:
- Experience（経験）: 実際に使用した経験や事例を紹介
- Expertise（専門性）: 統計や専門知識を示す
- Authoritativeness（権威性）: 信頼できる情報源を引用
- Trustworthiness（信頼性）: 情報を正確に記述

### 動画コンテンツの作り方

**一人語り形式**:
- 視聴者を引きつけるフック（5〜10秒）
- 明確なオープニング（「こんにちは〜」）
- 本論に入る前の予告（「本日のテーマは〜」）
- 結論はPREP法で構成

**ゆっくり動画形式**:
- 霊夢と魔理沙の掛け合いでわかりやすく解説
- 「霊夢:」と「魔理沙:」の対話形式
- 難しいスピード（1分程度の音声）
- 視聴者が理解しやすい専門用語を避ける

**サンプル sitemap.xml**:
```xml
<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url>
    <loc>https://yourdomain.com/post1</loc>
    <lastmod>2025-01-18</lastmod>
    <changefreq>weekly</changefreq>
  </url>
  <url>
    <loc>https://yourdomain.com/post2</loc>
    <lastmod>2025-01-18</lastmod>
    <changefreq>monthly</changefreq>
```

## 📞 よくある質問

### Q1: どれくらいで収入が出ますか？
**A**: 個人目標では、開始6ヶ月〜1年目で月収入3〜10万円を目指します。コンテンツの質と量によりますが、1年目で月収入10〜30万円も可能です。

### Q2: どのような記事を書くべきですか？
**A**:
- 自分の経験や知見を活かせるトピック
- 検索需要の大きいニッチトピック
- 実際に役に立つ「ハウツー」や「方法」系の記事
- E-E-A-Tを意識した内容

### Q3: 動画はどのような形式が良いですか？
**A**:
- 10〜15分の短い動画（完結に見きやすい）
- スライド形式か字幕付き
- アニメーションがあると良い

### Q4: AdSenseの審査に落ちてしまったらどうすればよいですか？
**A**:
- コンテンツの品質を見直す（文字数、構成、オリジナル性）
- 詐当な記事を削除・改善する
- 3ヶ月後に再申請する
